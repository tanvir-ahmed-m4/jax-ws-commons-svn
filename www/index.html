Project Description

Welcome to the JAX-WS commons project. This is a part of Project Metro in the Glassfish community at java.net.

This project aims to:

  • collect convenient utility code around JAX-WS RI
  • collect useful plugins and extensions around the JAX-WS RI (until they grow out of this project)

Overview

  • Status:POC
  • Licenses:Each extension has its own

Projects

  1. HTTP Session-scope service
  2. Thread scope service
  3. Spring support
  4. JAX-WS Maven2 plugin
  5. JSON Encoding
  6. SMTP transport
  7. JAX-WS Beans runtime-generator
  8. In-VM transport
  9. DIME Encoding
