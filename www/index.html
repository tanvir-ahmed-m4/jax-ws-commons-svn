Project Description

Welcome to the JAX-WS commons project. This is a project in the Web Services and XML community at java.net.

This project aims to:

  • collect convenient utility code around JAX-WS RI
  • collect useful plugins and extensions around the JAX-WS RI (until they grow out of this project)

Projects

  1. HTTP Session-scope service
  2. Thread scope service
  3. Spring support
  4. JAX-WS Maven2 plugin

Join us!

If you are interested in hosting your utility/plug-in/tool etc, please write to dev@jax-ws.dev.java.net so that we can arrange things. See this document for more about how to host your plugin.
