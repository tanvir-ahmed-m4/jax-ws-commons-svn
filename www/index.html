Project Description

Welcome to the JAX-WS 2.0 commons project. This is a project in the JWSDP subcommunity of the Java community on Web Services and XML at java.net.

This project aims to:

  • collect convenient utility code around JAX-WS 2.0
  • collect useful plugins and extensions around the JAX-WS RI 2.0 (until they grow out of this project)

Projects

  1. HTTP Session-scope service
  2. Spring support

Join us!

If you are interested in hosting your utility/plug-in/tool etc, please write to dev@jax-ws.dev.java.net so that we can arrange things. See this document for more about how to host your plugin.
