What is this?

JAX-WS RI implemented Endpoint API using lighte weight httpserver API. Light weight httpserver SPI allows to plugin any http server implementation for the API. This project implements the light weight httpserver API using grizzly. That means JAX-WS RI code doesn't have to change. And Endpoint.publish() gets all the benefits of grizzly(for e.g performance).
