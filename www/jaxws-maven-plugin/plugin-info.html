Plugin documentation

Goals available:

GoalDescription
jaxws:wsimportParses wsdl and binding files and generates Java code needed to access it.
jaxws:wsgenReads a JAX-WS service endpoint implementation class and generates all of the portable artifacts for a JAX-WS web service.
jaxws:wsgen-testReads a JAX-WS service endpoint implementation class and generates all of the portable artifacts for a JAX-WS web service (into the generate test source directroy.)

Due to MNG-1508, this requires 2.0.5 or higher.
