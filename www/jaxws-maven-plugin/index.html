jaxws-maven-plugin

Goals available:

Goal Description
jaxws:wsgen

A Maven 2 plugin which reads a service endpoint implementation class and generates all of the portable artifacts for a JAX-WS web service.

jaxws:wsimport A Maven 2 plugin which parses wsdl and binding files and produces a corresponding object model based on the JAXWS WsImport parsing engine.
