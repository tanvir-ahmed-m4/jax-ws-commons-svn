jaxws:wsimport

A Maven 2 plugin which parses wsdl and binding files and produces a corresponding object model based on the JAXWS WsImport parsing engine.

Parameters for the goal:

Parameter Type Expression Default Value Description
bindingDirectory (Optional) File - ${basedir}/src/jaxws Directory containing binding files.
bindingFiles (Optional) List - - List of files to use for bindings.If not specified, all .xml files in the bindingDirectory will be used.
catalog (Optional) File - - Catalog file to resolve external entity references support TR9401, XCatalog, and OASIS XML Catalog format.
destDir (Optional) File - ${project.build.outputDirectory} Specify where to place output generated classes Set to "" to turn it off
extension (Optional) boolean - false Allow to use the JAXWS Vendor Extensions.
httpproxy (Optional) String - - Set HTTP/HTTPS proxy. Format is [user[:password]@]proxyHost[:proxyPort]
keep (Optional) boolean - false Keep generated files.
packageName (Optional) String - - The package in which the source files will be generated.
pluginArtifactMap (Optional) Map ${plugin.artifactMap} - Map of of plugin artifacts.
project (Optional) MavenProject ${project} - No description.
sourceDestDir (Optional) File - ${project.build.directory}/jaxws/wsimport/java Specify where to place generated source files, keep is turned on with this option.
staleFile (Optional) File - ${project.build.directory}/jaxws/stale/.staleFlag The location of the flag file used to determine if the output is stale.
target (Optional) String - - Generate code as per the given JAXWS specification version. Setting "2.0" will cause JAX-WS to generate artifacts that run with JAX-WS 2.0 runtime.
verbose (Optional) boolean - false Output messages about what the tool is doing
wsdlDirectory (Optional) File - ${basedir}/src/wsdl Directory containing wsdl files.
wsdlFiles (Optional) List - - List of files to use for wsdls. If not specified, all .wsdl files in the wsdlDirectory will be used.
wsdlLocation (Optional) String - - No description.
wsdlUrls (Optional) List - - List of external wsdl urls to be compiled.
